# Basic Socket.io Demo

A bare minimum Socket.io implementation. Learn about [Websockets & Socket.io](https://youtu.be/1BfCnjr_Vjg) on Youtube. 

## Run it

```
cd server
npm install
npm start
```

Open the `app/index.html` file in a browser.
